## 1. Latihan : Lambda
Ubahlah kode dibawah ini menjadi fungsi menggunakan lambda. <p>

```bash
import math
def a(x):
    return x**2

def b(x, y)
    return math.sqrt(x**2 + y**2)  

def c(*args):
    return sum(args)/len(args)

def d(s):
    return "".join(set(s))
```

<li> Penjelasan </li>

1. Untuk memperluas daftar fungsi matematika gunakan import math <p>

2. Fungsi lambda yang menggunakan variabel a-d <p>


```bash
def a(x):
    return x**2
    a = lambda x : x ** 2
print(a(2))

def b(x, y):
    return math.sqrt(x**2 + y**2)
    b = lambda x, y : x ** 2  + y ** 2
print(b(2, 2))

def c(*args):
    return sum(args)/len(args)
    c = lambda *args : sum(args)/len(args)
print(c(5, -1, 8, 19))

def d(s):
    return "".join(set(s))
    d = lambda s: "".join(set(s))
print(d("Titan"))
```

3. Pemanggilan function lambda

```bash
print(a(2))
print(b(2, 2))
print(c(5, -1, 8, 19))
print(d("Titan"))
```
